# Lab 7 – VLAN Configuration

## Objective
Configure VLANs on a Cisco switch in Packet Tracer to segment a network into separate broadcast domains, improving security and traffic management.

## What I Did
- Created and named VLANs on a Cisco switch
- Assigned switch ports to the appropriate VLANs
- Verified VLAN configuration and connectivity between devices on the same VLAN
- Confirmed traffic isolation between devices on different VLANs
